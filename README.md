# El Pollo Loco

A browser-based jump-and-run game built with vanilla JavaScript and HTML Canvas.

## Features

- Side-scrolling platform gameplay
- Multiple levels
- Enemy AI
- Endboss fight
- Collectibles
- Throwable bottles
- Sound system with volume control
- Fullscreen mode
- Responsive mobile controls

## Technologies

- HTML5
- CSS3
- JavaScript (ES6 Modules)
- HTML Canvas API
- JSDoc
